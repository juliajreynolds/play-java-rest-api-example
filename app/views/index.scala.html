Play REST API - A coding sample from Julia Reynolds

Per your instructions: The example application has the concept of “blog posts”. Extend your application to support the concept of “authors”, including a new endpoint that displays a small blurb about the author of the blog posts. Unit tests and supporting documentation are encouraged.

To generate some starting data run the following using the httpie tool to post JSON:
Posts:
http POST localhost:9000/v1/posts title="Cooking with Julia" body="Vegetarian recipes."
http POST localhost:9000/v1/posts title="Hiring 101" body="Hire Julia"
Authors:
http POST localhost:9000/v1/authors name="Steven King" body="He writes scary books."
http POST localhost:9000/v1/authors name="Joshua Block" body="He writes good Java books."

To view the API directly:

  • /v1/authors
  • /v1/posts
